## Store Assets Structure

Each app folder contains:
- `short_description.txt` - 80 character short description
- `full_description.txt` - Full app description (4000 chars max)
- `privacy_policy.txt` - Privacy policy content
- `app-release.aab` - Signed AAB release file
- `privacy_policy_url.txt` - WordPress post URL (after creation)

## WordPress Privacy Policies

**Note:** WordPress authentication needs to be verified. The app password format may need adjustment or REST API permissions may need to be enabled.

To create privacy policy posts manually:
1. Log into WordPress admin at https://gloven.org/wp-admin
2. Create a new post for each app
3. Use the content from `privacy_policy.txt`
4. Set the URL slug to: `privacy-[app-slug]`
5. Save the post URL to `privacy_policy_url.txt`
